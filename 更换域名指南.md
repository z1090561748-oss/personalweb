# 🌐 更换域名指南

## 方法一：使用 Vercel 自定义域名（推荐）

### 步骤 1：进入域名设置

1. 访问：https://vercel.com
2. 登录你的账号
3. 进入项目 `personalweb`
4. 点击顶部的 **"Settings"** 标签
5. 在左侧菜单中找到 **"Domains"**
6. 点击进入

### 步骤 2：添加域名

1. 在 "Domains" 页面，你会看到一个输入框
2. 输入你想要使用的域名，例如：
   - `hover-web.com`
   - `www.hover-web.com`
   - `yourdomain.com`
3. 点击 **"Add"** 或 **"Add Domain"** 按钮

### 步骤 3：配置 DNS 记录

Vercel 会显示需要配置的 DNS 记录，通常有两种方式：

#### 方式 A：使用 CNAME 记录（推荐）

如果你的域名支持 CNAME：

1. **记录类型**: `CNAME`
2. **主机记录**: `@` 或 `www`（取决于你想用哪个）
3. **记录值**: `cname.vercel-dns.com.`（Vercel 会提供具体值）

#### 方式 B：使用 A 记录

如果必须使用 A 记录：

1. **记录类型**: `A`
2. **主机记录**: `@`
3. **记录值**: Vercel 提供的 IP 地址（通常是 `76.76.21.21`）

### 步骤 4：在域名注册商配置 DNS

1. 登录你的域名注册商（如 GoDaddy、Namecheap、阿里云等）
2. 找到 DNS 管理或域名解析设置
3. 添加 Vercel 提供的 DNS 记录
4. 保存设置

### 步骤 5：等待 DNS 生效

- DNS 生效通常需要 **几分钟到几小时**
- Vercel 会自动检测，当 DNS 配置正确时会显示 ✅

### 步骤 6：更新环境变量

域名生效后，需要更新环境变量：

1. 在 Vercel 项目设置中，进入 **"Environment Variables"**
2. 找到 `NEXT_PUBLIC_SITE_URL`
3. 点击编辑
4. 将值改为你的新域名（如 `https://hover-web.com`）
5. 点击 **"Save"**
6. 返回 **"Deployments"**，重新部署（Redeploy）

## 方法二：使用子域名

如果你想使用子域名（如 `web.yourdomain.com`）：

1. 在 Vercel Domains 中添加 `web.yourdomain.com`
2. 配置 DNS：
   - **类型**: `CNAME`
   - **主机**: `web`
   - **值**: `cname.vercel-dns.com.`
3. 等待生效
4. 更新环境变量

---

## 方法三：购买新域名

如果你还没有域名，可以：

### 选项 A：在 Vercel 购买

1. 在 Vercel Domains 页面
2. 点击 **"Buy Domain"**
3. 搜索并购买域名
4. Vercel 会自动配置

### 选项 B：在其他平台购买

推荐平台：
- **Namecheap**: https://www.namecheap.com
- **GoDaddy**: https://www.godaddy.com
- **Cloudflare**: https://www.cloudflare.com/products/registrar/
- **阿里云**: https://wanwang.aliyun.com（国内）

购买后，按照上面的步骤在 Vercel 中添加域名。

## 常见域名配置示例

### 示例 1：主域名（yourdomain.com）

**DNS 配置：**
```
类型: A
主机: @
值: 76.76.21.21
```

### 示例 2：www 子域名（www.yourdomain.com）

**DNS 配置：**
```
类型: CNAME
主机: www
值: cname.vercel-dns.com.
```

### 示例 3：同时支持主域名和 www

需要配置两条记录：
```
类型: A, 主机: @, 值: 76.76.21.21
类型: CNAME, 主机: www, 值: cname.vercel-dns.com.
```

## 验证域名配置

### 检查 DNS 是否生效

1. 使用在线工具检查：
   - https://dnschecker.org
   - https://www.whatsmydns.net

2. 在命令行检查：
```bash
# Windows
nslookup yourdomain.com

# 或使用 ping
ping yourdomain.com
```

### 检查 Vercel 状态

在 Vercel Domains 页面，域名旁边应该显示：
- ✅ **Valid Configuration** - 配置正确
- ⏳ **Pending** - 等待 DNS 生效
- ❌ **Invalid Configuration** - 配置有误

## 常见问题

### Q: DNS 配置后多久生效？
A: 通常 5 分钟到 2 小时，最长可能需要 48 小时。

### Q: 可以同时使用多个域名吗？
A: 可以！在 Vercel 中可以添加多个域名，它们都会指向同一个网站。

### Q: 如何设置 HTTPS？
A: Vercel 会自动为所有域名配置免费的 SSL 证书（HTTPS），无需手动配置。

### Q: 旧域名还能访问吗？
A: 可以！`personalweb-omega-six.vercel.app` 会继续工作，但建议使用新域名。
