# 首页配置说明

## 如何添加更多文档到首页

### 步骤 1：编辑配置文件

打开 `config/pages.json` 文件，添加更多文档：

```json
[
  {
    "id": "2d974318f7ec8063ab63ecd5a7ecddbe",
    "title": "HOVER web",
    "description": "新媒体艺术项目"
  },
  {
    "id": "另一个页面ID（32位十六进制）",
    "title": "作品标题（可选，会从 Notion 自动获取）",
    "description": "作品描述（可选）"
  }
]
```

### 步骤 2：获取 Notion 页面 ID

从 Notion 页面 URL 中提取：
- URL: `https://www.notion.so/Page-Title-2d974318f7ec8063ab63ecd5a7ecddbe`
- ID: `2d974318f7ec8063ab63ecd5a7ecddbe`（最后的 32 位字符）

### 步骤 3：确保页面已连接集成

每个 Notion 页面都需要：
1. 打开页面
2. 点击右上角 "..." → "连接"
3. 选择你的集成

### 步骤 4：保存并部署

1. 保存 `config/pages.json` 文件
2. 提交代码：
   ```bash
   git add config/pages.json
   git commit -m "添加新作品"
   git push
   ```
3. Vercel 会自动重新部署

## 配置说明

### 字段说明

- **id**（必需）：Notion 页面的 32 位 ID
- **title**（可选）：显示标题，如果不提供会从 Notion 自动获取
- **description**（可选）：作品描述，显示在卡片下方

### 示例配置

```json
[
  {
    "id": "2d974318f7ec8063ab63ecd5a7ecddbe",
    "title": "HOVER web",
    "description": "新媒体艺术项目"
  },
  {
    "id": "abc123def45678901234567890123456",
    "title": "数字艺术",
    "description": "交互式数字艺术作品"
  },
  {
    "id": "def456abc12378901234567890123456",
    "description": "实验性视觉项目"
  }
]
```

## 常见问题

### Q: 添加文档后不显示？
A: 
1. 检查页面 ID 是否正确
2. 确认页面已连接到集成
3. 查看浏览器控制台的错误信息

### Q: 标题显示不正确？
A: 
1. 检查 Notion 页面的标题属性
2. 可以在配置中手动设置 title

### Q: 如何排序？
A: 
在 `pages.json` 中的顺序就是显示顺序，调整数组顺序即可。
