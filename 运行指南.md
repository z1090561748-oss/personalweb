# 如何运行网站

## 方法一：开发模式（推荐）

### 步骤：

1. **打开终端/命令行**
   - Windows: 按 `Win + R`，输入 `cmd` 或 `powershell`，回车
   - 或者在项目文件夹中，按住 `Shift` 键，右键点击空白处，选择"在此处打开 PowerShell 窗口"

2. **进入项目目录**
   ```bash
   cd g:\notion
   ```

3. **启动开发服务器**
   ```bash
   npm run dev
   ```

4. **等待启动完成**
   - 看到类似这样的输出表示成功：
     ```
     ▲ Next.js 13.5.6
     - Local:        http://localhost:3000
     - Ready in 2.3s
     ```

5. **打开浏览器访问**
   - 访问：http://localhost:3000
   - 或者点击终端中显示的链接

6. **停止服务器**
   - 在终端中按 `Ctrl + C`

---

## 方法二：生产模式

### 步骤：

1. **构建项目**
   ```bash
   npm run build
   ```

2. **启动生产服务器**
   ```bash
   npm start
   ```

3. **访问网站**
   - 浏览器访问：http://localhost:3000

---

## 快速命令参考

```bash
# 开发模式（热重载，修改代码自动刷新）
npm run dev

# 构建生产版本
npm run build

# 启动生产服务器
npm start

# 代码检查
npm run lint
```

---

## 注意事项

1. **确保已安装依赖**
   - 如果第一次运行，先执行：`npm install`

2. **确保 Token 已配置**
   - 检查 `.env.local` 文件是否存在
   - 文件内容应为：`NOTION_API_TOKEN=你的token`

3. **端口占用**
   - 默认使用 3000 端口
   - 如果被占用，Next.js 会自动使用 3001、3002 等

4. **修改代码后**
   - 开发模式下会自动刷新
   - 如果修改了 `.env.local`，需要重启服务器
