# Notion 数据库设置指南

## 步骤 1：创建 Notion 数据库

### 1.1 创建新页面
1. 在 Notion 中创建新页面
2. 命名为 "作品集" 或 "Portfolio"

### 1.2 添加数据库
1. 在页面中输入 `/database` 或 `/table`
2. 选择 "Table - Inline" 或 "Table - Full page"
3. 创建数据库

### 1.3 设置数据库属性

数据库需要包含以下列（属性）：

| 属性名 | 类型 | 说明 | 必需 |
|--------|------|------|------|
| **Name** | Title | 作品名称（自动创建） | ✅ |
| **Page ID** | Text | Notion 页面 ID（32位） | ✅ |
| **Cover** | Files & media | 展示图片 | ✅ |
| **Description** | Text | 作品描述 | ❌ |
| **Category** | Select | 作品分类 | ❌ |
| **Date** | Date | 创建日期 | ❌ |

### 1.4 添加作品数据

在数据库中添加每一行作品：

1. **Name（名称）**：输入作品名称，如 "HOVER web"
2. **Page ID**：输入对应 Notion 页面的 ID（32位，不含连字符）
   - 例如：`2d974318f7ec8063ab63ecd5a7ecddbe`
3. **Cover（封面）**：上传展示图片
4. **Description（描述）**：输入作品描述（可选）
5. **Category（分类）**：选择分类（可选）
6. **Date（日期）**：选择日期（可选）

### 1.5 获取数据库 ID

1. 点击数据库右上角的 "..." 菜单
2. 选择 "Copy link"
3. 从链接中提取数据库 ID：
   - 链接格式：`https://www.notion.so/数据库标题-{32位ID}`
   - 例如：`https://www.notion.so/Portfolio-abc123def456...`
   - ID 就是最后的 32 位字符

### 1.6 连接集成

1. 点击数据库右上角的 "..." 菜单
2. 选择 "连接" (Connections)
3. 选择你的 Notion 集成

---

## 步骤 2：配置网站

### 2.1 更新配置文件

在 `config/pages.json` 中，只需要配置数据库 ID：

```json
{
  "databaseId": "你的数据库ID（32位）"
}
```

### 2.2 网站会自动读取

网站会自动：
- ✅ 从数据库读取所有作品
- ✅ 显示作品名称和图片
- ✅ 点击后跳转到对应页面

---

## 注意事项

1. **Page ID 格式**：必须是 32 位十六进制，不含连字符
2. **图片格式**：支持 JPG、PNG、WebP 等
3. **数据库权限**：确保数据库已连接到你的集成
4. **页面权限**：确保每个作品页面也已连接到集成
